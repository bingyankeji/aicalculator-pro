===============================================
  AI Calculator - 快速启动说明
===============================================

📁 项目目录: C:\Users\Administrator\Desktop\test

🚀 启动方式：

方式1: 快速启动（推荐）
  双击: 快速启动.bat
  - 自动检查依赖
  - 自动打开浏览器
  - 启动开发服务器

方式2: 完整启动
  双击: 启动项目.bat
  - 显示详细信息
  - 手动打开浏览器
  - 端口: 3001

方式3: 手动启动
  1. 双击: 安装依赖.bat（首次运行）
  2. 双击: 启动项目.bat

===============================================

📦 首次使用：

1. 确保已安装 Node.js 18+
   下载地址: https://nodejs.org/

2. 双击 "安装依赖.bat" 安装所需包

3. 双击 "快速启动.bat" 启动项目

4. 浏览器访问: http://localhost:3001

===============================================

🔧 其他工具：

- 安装依赖.bat       - 安装/重装项目依赖
- 启动项目.bat       - 启动开发服务器（详细信息）
- 快速启动.bat       - 快速启动（自动打开浏览器）
- 构建生产版本.bat   - 构建生产版本

===============================================

⚙️ 端口配置：

默认端口: 3001
访问地址: http://localhost:3001

如需修改端口，编辑 package.json:
  "dev": "next dev -p 端口号"

===============================================

❓ 常见问题：

Q: 提示端口被占用？
A: 修改 package.json 中的端口号

Q: npm install 很慢？
A: 使用国内镜像：
   npm config set registry https://registry.npmmirror.com

Q: 启动失败？
A: 1. 检查 Node.js 版本 >= 18
   2. 删除 node_modules 重新安装
   3. 查看错误日志

===============================================

📝 开发命令（在命令行中）：

npm run dev         - 启动开发服务器
npm run build       - 构建生产版本
npm run start       - 启动生产服务器
npm run lint        - 代码检查
npm run type-check  - 类型检查

===============================================

💡 提示：

- 首次启动需要安装依赖，可能需要几分钟
- 保存代码后会自动热重载
- 按 Ctrl+C 可停止服务器
- 建议使用 Chrome 浏览器开发
